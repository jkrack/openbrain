# Work Day / Weekend Awareness — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** OpenBrain adapts its personality and skill scheduling based on whether today is a work day or weekend, with configurable work days.

**Architecture:** A `getDayMode()` utility checks today's day against `settings.workDays`. System prompt loading picks the day-appropriate file. Scheduler and trigger handlers gate skills by `dayMode`. Onboarding asks users to set their work days.

**Tech Stack:** TypeScript, Obsidian Plugin API, Vitest

**Spec:** `docs/superpowers/specs/2026-03-25-day-mode-design.md`

```typescript
// src/dayMode.ts

/**
 * Determine if today is a work day or weekend based on user's configured work days.
 * @param workDays Array of day numbers (0=Sun, 1=Mon, ..., 6=Sat)
 */
export function getDayMode(workDays: number[]): "work" | "weekend" {
  const today = new Date().getDay();
  return workDays.includes(today) ? "work" : "weekend";
}
```

Replace the existing `loadSystemPrompt()` method:

```typescript
  private async loadSystemPrompt(): Promise<void> {
    const mode = getDayMode(this.settings.workDays);
    const dayPath = mode === "work"
      ? "OpenBrain/system-prompt-work.md"
      : "OpenBrain/system-prompt-weekend.md";
    const fallbackPath = "OpenBrain/system-prompt.md";

    // Try day-specific file first, then fallback
    for (const path of [dayPath, fallbackPath]) {
      const file = this.app.vault.getAbstractFileByPath(path);
      if (file instanceof TFile) {
        const content = await this.app.vault.read(file);
        if (content.trim()) {
          this.settings.systemPrompt = content.trim();
          return;
        }
      }
    }

    // No file found — seed the fallback
    try {
      await this.app.vault.create(fallbackPath, this.settings.systemPrompt);
    } catch { /* folder may not exist yet */ }
  }
```
